# Cybersecurity Portfolio

## About

This repository documents my cybersecurity learning journey through practical labs, traffic analysis, threat hunting investigations, and security reports.

The objective of this portfolio is to develop real-world skills in:

* Blue Team Operations
* Security Monitoring
* Threat Hunting
* Network Traffic Analysis
* Incident Response
* Log Analysis
* Security Documentation

## Projects

### 1. Beaconing and Command & Control (C2) Detection

Analysis of network traffic to identify beaconing patterns, persistence mechanisms, and potential command-and-control communications.

### 2. Threat Hunting Investigation

Investigation of suspicious network entities through traffic analysis, OSINT techniques, and behavioral correlation.

### 3. Traffic Timeline Reconstruction

Reconstruction of complete browsing sessions using DNS, QUIC, TLS, and network correlation techniques.

### 4. Network and Log Correlation

Correlation between Windows logs, active processes, PowerShell output, and network traffic to identify potential anomalies.
